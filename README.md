**Course description:
**

The Wikipedia Race: Building (Artificially) Intelligent Search Agents

How will your AI agent start with Banana and finish at Quantum Mechanics using only Wikipedia links? In this lab, students will build AI agents that "race" across Wikipedia by clicking links from one page to the next, trying to reach the target article in the fewest possible steps. The twist: at each step, the agent only sees the current page and its links. No full map allowed! Students will experiment with different strategies for choosing the "best" next link, using patterns in text, structure, and logic to guide their decisions. Along the way, they will explore how large networks of information are organized and how algorithms can navigate them efficiently. The course culminates in a live tournament where students' agents compete to solve a variety of increasingly challenging races. Success will depend on creativity, strategy, and a deep understanding of how to search intelligently in complex systems.
